# Sorting Algorithms & Big O
Project done during **ALX Software Engineering Scholarship 2022** at **Alx Students Education**. It aims to learn about different sorting algorithms, Big O notation, and how to evaluate the time complexity of an algorithm in **C language**.

## Technologies
* C files are compiled using `gcc 9.8.4`
* Tested on Ubuntu 20.04 LTS
* Scripts written in Bash 4.4.23(2)

## Files
